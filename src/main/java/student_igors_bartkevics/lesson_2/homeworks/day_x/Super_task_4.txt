Каким языком является Java: тиризированным или не типизированным?

Java является типизированным языком программирования.
Типизация делится на несколько категорий:
    - Во-первых, Java является языком программирования со СТАТИЧЕСКОЙ типизацией,
    где конечные типы переменных устанавливаются на этапе компиляции;
    - Во-вторых типизация в Java - СИЛЬНАЯ, то есть Java не позволяет смешивать в выражениях
    различные типы и не выполняет автоматические неявные преобразования, например нельзя
    вычесть из строки множество;
    - В-третьих, типизация в Java ЯВНАЯ, то есть тип новых переменных / функций / их аргументов
    нужно задавать явно. Соответственно языки с неявной типизацией перекладывают эту задачу на
    компилятор / интерпретатор.
